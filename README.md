# 🌿 Fazenda Recreio

Projeto de site para a Fazenda Recreio - um espaço de lazer, eventos e contato com a natureza.

## 📋 Sobre o Projeto

Este site foi desenvolvido para apresentar a Fazenda Recreio, um local destinado a eventos, lazer e atividades ao ar livre. O projeto tem como objetivo fornecer informações sobre a fazenda, seus serviços, estrutura e formas de contato.

## 🚀 Tecnologias Utilizadas

- **HTML5** - Estrutura do site
- **CSS3** - Estilização e design responsivo
- **JavaScript** - Interatividade e funcionalidades
- **Git/GitHub** - Controle de versão e hospedagem
